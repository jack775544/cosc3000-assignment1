About this Visualisation

This visualisation is a heatmap of all the public transport stops in the South East Queensland Region

It allows for a high level overview of the entire public transport network or to drill down on a per stop basis

To see more detailed information on a single stop, zoom the map in so markers appear on the map. Then click on the individual markers for more info

Changing to cluster view will alter markers that are displayed on the map so that the points are clustered together in logical groups using spatial data clustering
